3. 视觉风格（Visual Style Guide）
🎨 色彩（Color Palette）
背景：深色渐变（深蓝 / 深绿 / 深灰）

主色：青绿（科技感）

强调色：柔和金色或青蓝光效

文本：白色 + 轻微透明度（rgba(255,255,255,0.85)）

🧱 布局（Layout）
大留白

大标题

居中布局

极简线条

不使用复杂边框

不使用花哨动画

✨ Three.js 风格（3D Style）
抽象科技感

粒子、线框、节点网络

柔和光效

轻微相机跟随鼠标

不喧宾夺主，保持克制

5. Three.js 场景开发文档（3D Scene Requirements）
🎯 目标
构建一个抽象科技感、自然流动、轻量级的 Three.js 场景作为 Hero 背景。

🌌 场景元素（Scene Elements）
中央：缓慢旋转的多面体（低面数）

周围：节点网络（Points + Lines）

背景：粒子（星光效果）

光照：柔和点光源 + 环境光

相机：轻微跟随鼠标（parallax）

📱 移动端降级
减少粒子数量

移除节点网络

或使用静态背景图

⚙ 性能要求
FPS ≥ 50

模型 < 1MB

粒子数量 < 2000

使用 requestAnimationFrame
